Introduction

Welcome to this entry level introduction to html. To start of with what does html even stand for and what can we do with it? html stands for Hyper Text MarkUp Language and is the standard language used in creating web pages. It is a way of annotating webpages so that computers can interpert and display them properly, by using a collection of different tags you can essentially build a static webpage from start to finish.

HTML has had multiple iterations since the language has been developed such as html 2, 3, 4 and now the current version html 5.

Examples of common tags

  • <body> This tag is used for containing all the visible content that will be displayed on the webpage, anything from text, videos, links and headings.
  • <header>We use the heading tag to define the heading of the webpage, it provides the introductory information normally presented at the top of a webpage
  • <h1> so the h1 tag can range all the way from h1 to h6 in descending order to create subheaders for your content varying in size. It is fundementally different from the header tag as that is the main header for the entire page
  • <p> The p tag is used for when you want to create paragraphs of text on your webpage
  • <section> Section tages can be used to group together bundles of information

    • Basics

    In this section we will cover over the basic tools that you will need to start any webpage and get it to function on a very basic level but will techinically function as a webpage.

    At the start of every html document you need to start off with <DOCTYPE html!> this is what helps the computer determine that it is dealing with a html document. There are various ways to go about presenting information on your webpage but normally you would use the "p" tag to present any text on the webpage and h1-h6 for any headings or subheaders.

    Attributes

    Attributes are a way for you to add content to an element wihtout it appearing on thr actual webpage itself, so for example adding a class to an element is an example of an attribute, as you are assigning a name to the element which would be the hidden content not displayed below is an example:

    <p class="dog">My dog is hungry<p>

    so in this example the class="dog" part would be the attribute as it is the hidden content that will not be displayed to the viewer of the webpage.

    Classes

    Classes are important because they can name elements that so you can specifically call on that element later on. They are flexible cause you can add more than one class to an element

    One of the main reasons that you would use a class is for styling with css or for selecing elements. Class can be made with letters, digits and hyphens but can never start with a digit and if you need to have a space you use an underscore.

    Colors

    Colors can be used in a html page by using inline css. There can be many different ways that you can make your colors with for example, you can use:

  • RGB values to specify your red, greens and blue to a varying degree of intensity to make your desired color.

  • Hexidecimal is another method of representing the color needed by using a combination of up to 6 alphanumeric characters following the hash symbol to create your color.
  • RGBAis very similar to the rgb method of creating colors on a webpage but with this method you can also control the opacity of the color by adding an alpha value which can range all the way from 0 which would be transparent to 1 which is opaque

    • Divs

    A div element requires no attributes but you will commonly find class and id attributes attributed to div elements, this is so you can call on them a lot later on.

    Div elements can also be used as a container so that you can group together entire sections of a webpage to style the entire section. You can also use multiple divs on a webpage to make multiple groups as is one of the most important elements needed when styling in css.
